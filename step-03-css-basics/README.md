# Step 3: CSS Basics

This is what your page should look like after adding colours, fonts, and button styling.

## What was added to style.css

- `body` — set the font, removed default margins, added a light background colour
- `h1` and `h2` — custom heading colours
- `p` — comfortable font size and line spacing
- `a` — styled links with colour and removed the default underline
- `button` — background colour, white text, padding, rounded corners, pointer cursor

## Key concepts

- **CSS rules** follow the pattern: `selector { property: value; }`
- **Colours** can be written as names (`teal`) or hex codes (`#028090`)
- `border-radius` rounds corners — higher values make them rounder
- `cursor: pointer` shows a hand icon when hovering over the button
- `text-decoration: none` removes the underline from links
