# Step 1: Environment Setup

This is your starting point. These two files are what you should have after setting up VS Code, installing Live Server, and creating your project folder.

## What to do

1. Copy `index.html` and `style.css` into your `my-website` folder
2. Right-click `index.html` in VS Code and select "Open with Live Server"
3. You should see "Hello, World!" in your browser

## What you will learn

- The basic structure of an HTML document
- How `<head>` and `<body>` work
- How to link a CSS file to HTML
