# Final Project: Complete Portfolio Page

This is the finished page with everything from the workshop combined. If you want to start fresh or check your work, copy these two files into your project folder.

## What is included

- **HTML** — semantic structure with header, nav, main, section, cards, and footer
- **CSS** — fonts, colours, button styling, box model spacing, Flexbox layout, hover effects with transitions, and responsive design with a media query

## Features

- Styled header with horizontal navigation
- Three cards arranged side by side using Flexbox
- Smooth hover effects on cards and buttons
- Responsive layout that stacks on mobile screens
- Clean, accessible code using semantic HTML

## How to use

1. Copy `index.html` and `style.css` into your `my-website` folder
2. Open `index.html` with Live Server in VS Code
3. Try resizing your browser window to see the responsive design in action

## Ideas to keep building

- Change the colours to your own palette
- Add more cards with different content
- Add an image to each card
- Create a second page and link to it from the navigation
- Try using Google Fonts for a different typeface
- Add a "hero" section at the top with a large heading and background colour
