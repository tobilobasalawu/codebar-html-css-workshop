# Step 2: HTML Structure

This is what your page should look like after adding HTML elements and semantic tags.

## What was added

- A `<header>` with an `<h1>` title and `<nav>` with three links
- A `<main>` area with a `<section>` containing a heading, paragraph, image, and button
- A `<footer>` with a paragraph
- Semantic tags (`header`, `nav`, `main`, `section`, `footer`) instead of plain `div` elements

## Key concepts

- **Semantic HTML** gives your page meaning — browsers and screen readers can understand the purpose of each section
- `<a href="#">` creates a link (the `#` is a placeholder URL)
- `<img>` needs a `src` (the image address) and `alt` (a description for accessibility)
- The page will look plain at this point — that is expected! CSS comes next
