# Step 4: Box Model & Spacing

This is what your page should look like after adding margin, padding, and border styles.

## What was added to style.css

- `header` — teal background, white text, padding inside
- `header h1` — overrides the heading colour to white (since it is inside the coloured header)
- `section` — padding inside, centred on the page with `margin: auto`, max width of 800px
- `footer` — dark background, white text, centred text

## Key concepts

- **Padding** adds space *inside* an element (between the content and the border)
- **Margin** adds space *outside* an element (between the element and its neighbours)
- `padding: 20px 40px` is shorthand — first value is top/bottom, second is left/right
- `margin: 20px auto` centres the element horizontally (`auto` splits the remaining space equally)
- `max-width` stops content from stretching too wide on large screens
