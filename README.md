# HTML & CSS for Beginners — codebar Festival 2026

Welcome to the **HTML & CSS for Beginners** workshop! This repo contains all the code from the session so you can follow along, catch up if you fall behind, or revisit everything afterwards.

## How to Use This Repo

Each folder matches a section of the workshop. If you get stuck at any point, just open the matching folder and copy the code into your project.

| Folder | Section | What You Will Learn |
|--------|---------|---------------------|
| `step-01-setup` | Environment Setup | Your starting HTML and CSS files |
| `step-02-html-structure` | HTML Fundamentals | Headings, paragraphs, links, images, semantic tags |
| `step-03-css-basics` | CSS Fundamentals | Selectors, colours, fonts, button styling |
| `step-04-box-model` | Box Model & Spacing | Margin, padding, border, centring content |
| `step-05-flexbox` | Layout with Flexbox | Cards, navigation, side-by-side layouts |
| `step-06-layouts` | Flexbox Layouts | Div containers |
| `step-07-hover-effects` | Interactive Styling | Hover effects, transitions, shadows |
| `step-08-responsive` | Responsive Design | Media queries, mobile-friendly layouts |
| `final-project` | Complete Project | The finished page with everything combined |

## Getting Started

### What You Need

- A computer (Windows, macOS, or Linux)
- [Visual Studio Code](https://code.visualstudio.com/) (free code editor)
- The **Live Server** extension for VS Code

### Quick Setup

1. Download and install [VS Code](https://code.visualstudio.com/)
2. Open VS Code and install the **Live Server** extension:
   - Click the Extensions icon in the left sidebar (four small squares)
   - Search for "Live Server"
   - Click Install on "Live Server" by Ritwick Dey
3. Create a folder on your Desktop called `my-website`
4. Open that folder in VS Code (File > Open Folder)
5. Create two files: `index.html` and `style.css`
6. Right-click `index.html` in the sidebar and select "Open with Live Server"

## Project Structure

Your project folder should look like this:

```
my-website/
  index.html
  style.css
```

## Following Along

During the workshop, you will be writing code in your own `my-website` folder. This repo is your safety net — if something goes wrong or you fall behind, just:

1. Open the step folder that matches where we are in the workshop
2. Copy the code from `index.html` and `style.css` into your own files
3. Save and check your browser
